# 🩺 Health Monitoring System (Arduino + Tinkercad)

## 📌 Overview

The Health Monitoring System is an Arduino-based project designed to monitor basic human health parameters such as body temperature and pulse rate in real time. This project is created and simulated using Tinkercad, making it accessible without physical hardware.

It demonstrates how embedded systems can be applied in healthcare for continuous and efficient monitoring.

---

## 🎯 Objectives

* Monitor vital health parameters
* Display real-time sensor data
* Build a low-cost healthcare prototype
* Learn Arduino and sensor integration

---

## ⚙️ Components Used

* Arduino Uno
* Temperature Sensor (e.g., LM35)
* Pulse Sensor (or simulated input)
* LCD Display / Serial Monitor
* Resistors
* Jumper Wires

---

## 🔌 Working Principle

The sensors collect data from the user and send analog signals to the Arduino Uno. The Arduino processes this data and converts it into meaningful values like temperature (°C) and heart rate (BPM). The results are displayed in real time using the Serial Monitor or an LCD screen.

---

## 🖥️ Simulation

This project is designed using Tinkercad, where the circuit is built and tested virtually. It helps in understanding circuit behavior without requiring physical components.
